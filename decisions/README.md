# Decisions

Use this folder for durable decisions and rationale that future agents should respect.

## What To Put Here

- Repository conventions.
- Family planning decisions.
- Chosen defaults, such as providers, schools, tools, or workflows.
- Superseded decisions, clearly marked as superseded.

## What To Read From Here

Read this folder before changing structure, replacing established routines, or revisiting a topic that may already have a decision history.

## File Naming

Use date-first names such as `2026-07-06-repository-structure.md`.
