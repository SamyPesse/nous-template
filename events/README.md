# Events

Use this folder for dated family events, trips, milestones, holidays, birthdays, appointments, and meaningful occurrences.

## What To Put Here

- Event plans and summaries.
- Trip notes, packing context, itineraries, and post-event learnings.
- Birthday plans, holiday traditions, anniversaries, and family milestones.
- Meeting notes for important family decisions.

## What To Read From Here

Read this folder when planning a similar future event or when historical context matters. Past events often contain useful preferences and logistics that should be reused.

## File Naming

Prefer date-first filenames: `2026-07-06-family-dinner.md` or `2026-08-summer-trip.md`.
