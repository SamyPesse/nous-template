# Templates

Use this folder for reusable Markdown templates that keep the knowledge base consistent.

## What To Put Here

- Profile templates.
- Event templates.
- Decision record templates.
- Contact list templates.
- Any repeated structure that helps agents create new files cleanly.

## What To Read From Here

Agents should check this folder before creating a new recurring file type. Prefer adapting an existing template over inventing a new structure.

## Maintenance

Keep templates short and practical. A template should include only fields that are useful enough to maintain over time.
